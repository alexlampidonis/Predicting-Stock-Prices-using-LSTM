# Predicting-Stock-Prices-using-LSTM
Predicted the daily percent movement of 442 companies on the 1st of April 2022, using their daily percentage movement from 05/04/2010 to 31/03/2022. For this, an LSTM model was created, and also Optuna was used for appropriate hyperparameter optimization. Finally, Captum was used for some interpretation of the model's features.
